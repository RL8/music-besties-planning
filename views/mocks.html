UserRanking {

  userId: string,
  albumId: string,
  oldRank: number,
  newRank: number,
  timestamp: timestamp
}

Backend Implementation Details

API Endpoints

GET /api/friends
  - Returns user's friends list
  - Query params: limit, offset, search

GET /api/friend-requests
  - Returns pending friend requests
  - Query params: type (incoming, outgoing)

POST /api/friends/request
  - Send a friend request
  - Body: { userId }

PUT /api/friends/request/{requestId}
  - Accept or reject a friend request
  - Body: { status: 'accepted' | 'rejected' }

DELETE /api/friends/{friendId}
  - Remove a friend
Data Models

Friend {
  userId: string,
  username: string,
  name: string,
  avatarUrl: string,
  stats: {
    commonArtists: number,
    similarityScore: number
  },
  friendSince: timestamp
}

FriendRequest {
  id: string,
  fromUser: {
    id: string,
    username: string,
    name: string,
    avatarUrl: string
  },
  toUser: {
    id: string,
    username: string
  },
  status: 'pending' | 'accepted' | 'rejected',
  createdAt: timestamp
}
Controllers
  • FriendsController - Manages friend relationships
  • FriendRequestController - Handles friend requests
  • UserSearchController - Handles user search for finding friends

Backend Implementation Details

API Endpoints

GET /api/compare/{friendId}
  - Returns comparison data between user and friend
  - Query params: type (albums, artists, songs)

GET /api/compare/{friendId}/stats
  - Returns statistical comparison data
  - Includes match percentage, common items, etc.

POST /api/compare/share
  - Creates a shareable comparison link
  - Body: { friendId, type }
Data Models

ComparisonResult {
  userId: string,
  friendId: string,
  type: 'albums' | 'artists' | 'songs',
  stats: {
    matchPercentage: number,
    commonCount: number,
    differentCount: number,
    uniqueToUserCount: number,
    uniqueToFriendCount: number
  },
  items: ComparisonItem[]
}

ComparisonItem {
  id: string,
  title: string,
  artist: string,
  userRank: number | null,
  friendRank: number | null,
  status: 'match' | 'different' | 'unique-user' | 'unique-friend'
}
Controllers
  • ComparisonController - Handles comparison logic
  • ComparisonStatsController - Calculates statistics
  • ShareController - Manages sharing functionality
